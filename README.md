## 🚀 대표 프로젝트

### 1. Narratix | 프론트엔드 리드
> **취준생의 경험을 아카이빙하고 자기소개서 작성을 돕는 플랫폼.**

- **기술 스택**: React, TypeScript, TanStack Query, Tailwind CSS, FSD(Feature-Sliced Design), Vitest
- **주요 기여**:
  - 모듈성과 유지보수성 향상을 위해 **FSD Lite 아키텍처**로의 점진적 리팩토링을 주도했습니다.
  - 4-depth 중첩 URL 라우팅에서 **구독 훅을 Leaf Node로 분리**해 리렌더링 범위를 최소화했습니다.
  - URL·LocalStorage·State를 동기화하는 커스텀 `useSearch` 훅을 구현하고, **Vitest 단위 테스트**로 디바운싱·저장소 복원 동작을 검증했습니다.
  - 순수 JavaScript `Date` 기반 **커스텀 캘린더**를 개발해 외부 라이브러리 의존성을 제거하고 번들 사이즈를 줄였습니다.

  - **GitHub Actions 기반 CI/CD 파이프라인**을 구축해 배포를 자동화했습니다.

### 3. [Dinary](https://github.com/Sminp/Dinary) | 프론트엔드 개발

### 4. [3D Interactive Portfolio](https://github.com/Sminp/portfolio) | 개인 프로젝트
